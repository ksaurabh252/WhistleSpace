# WhistleSpace – Client (Frontend)

This is the frontend of **WhistleSpace**, a privacy-first anonymous feedback platform. Built with **React**, **Vite**, and **Chakra UI**, it allows users to submit feedback anonymously, and admins to manage and review submissions.

## 🚀 Features

- Anonymous feedback submission
- AI-powered sentiment & category tagging (via backend)
- Admin dashboard with pagination, search, and filters
- Responsive Chakra UI layout with mobile drawer navigation
- Light/dark theme toggle
- Confetti animation on successful submission

## 🧱 Tech Stack

- React + Vite
- Chakra UI
- Axios
- React Router DOM
- react-confetti + react-use
- Environment config via `.env`

## 📦 Installation

```bash
cd client
npm install

VITE_BACKEND_URL=http://localhost:5000

Run Dev Server
npm run dev
```
